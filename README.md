# Automation API – Text & Document AI Service

این پروژه یک سرویس **FastAPI** است که برای پردازش متن، فایل و PDF طراحی شده و چند قابلیت هوش مصنوعی و اتوماسیون را در یکجا ترکیب می‌کند.

هدف این سرویس:
- پردازش و تمیز کردن متن
- گرفتن آمار متن (تعداد کاراکتر و کلمات)
- ساخت گزارش‌ها (TXT / JSON / CSV)
- خلاصه‌سازی متن (Summarization)
- ترجمه متن (Translation)
- پردازش فایل متنی و PDF
- (اختیاری) تحلیل احساس متن (Sentiment Analysis) – اگر فعال شده باشد
